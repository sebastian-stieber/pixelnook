## What PixelNook Launcher does

- Local-first game library: scan your games folder and browse by system
- Console-like UI: smooth navigation, animated backgrounds, polished transitions
- Fully usable with **gamepad** + keyboard/mouse
- Favorites + recently played
- Quick Menu overlay for fast navigation + quick access
- Highly customizable layout:
  - Reorder/hide main pages (Home / Library / Guides / Settings)
  - Customize Home rows (recently added/played, favorites, per-system rows, widget grid)
  - Customize Quick Menu entries and recent-games mode
- Built-in guides (setup, troubleshooting, legal)

## Setup (first run)

1. **Set your Games Directory** in Settings ‚Üí Library
2. **Configure emulators** in Settings ‚Üí Emulators (set executable paths)
3. **Configure game systems** in Settings ‚Üí Game Systems  
   - Use **Auto-configure** if you want PixelNook to pick emulator/core defaults
4. Open **Library** and launch a game üéÆ

> PixelNook does not provide ROMs, BIOS files, or encryption keys.

## Included emulator support (v0.1.x)

PixelNook includes configuration support for:
- RetroArch (+ cores)
- DuckStation
- PCSX2
- Dolphin
- mGBA

More emulator/system configs will be added over time.
